# ssh-swap

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

CLI tool to quickly switch between multiple SSH keys (e.g. multiple GitHub accounts).

## Install

```bash
git clone git@github.com:AndyChang33/ssh-swap.git
cd ssh-swap
node bin/ssh-swap init   # create config, scan keys, add shell alias
```

`init` will:
1. Create `config.json`
2. Scan `~/.ssh/` and let you pick which keys to add
3. Add `alias ss=...` to your `~/.zshrc` or `~/.bashrc`

## Usage

```bash
ss                       # interactive menu (arrow keys / j,k)
ss use <name>            # switch: ssh-add -D → ssh-add <key> → ssh -T git@github.com
ss use                   # no arg → interactive menu
ss list                  # list all configured keys
ss current               # show currently loaded key
ss scan                  # scan ~/.ssh/ for new keys
ss add <name> <path>     # manually add a key
ss remove <name>         # remove a key
ss init                  # first-time setup
```

## Config

`config.json` (auto-generated by `init` / `scan`, or edit manually):

```json
{
  "keys": {
    "personal": { "path": "~/.ssh/id_ed25519", "email": "you@example.com" },
    "work": { "path": "~/.ssh/id_rsa_work", "email": "you@company.com" }
  }
}
```

## Requirements

- Node.js
